# Statistics

How to run this app:

Open a terminal and run (leave it open and running)

```
mongod
```

On a new terminal Go to project's directory.

```
cd Stati...
code .
```

Install (if needed)

```
npm install
```

Run

```
npm start
```

or watch (watches changes on save and restarts the server)

_note: this applies to app.js file only (for now) _

```
npm run watch
```
